MicSwitch

Switch your Mac’s microphone from the menu bar.

MicSwitch is a small macOS app that lives in your menu bar. It lets you see all available audio input devices (built-in mic, external USB mics, iPhone as mic, etc.) and switch the system input with one click.

Features

  • Menu bar access — Open the list from the menu bar icon; no need to open System Settings.
  • All inputs in one place — Built-in microphone, USB mics, and Continuity Camera (iPhone) inputs in a single list.
  • One-click switch — Tap a device to set it as the current system input.
  • Input volume — Adjust the input level of the current microphone from the same menu.
  • English & 简体中文 — App and UI follow your system language.

Requires macOS 13 or later. Available on the Mac App Store.
